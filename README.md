# Professional Personal Portfolio

A responsive personal portfolio website showcasing my cloud engineering projects, technical skills, and professional experience. Built with HTML, CSS, and JavaScript, and deployed on Vercel.

## Live Demo

[View Portfolio](https://khaya-professional-portfolio.vercel.app/)

## About

This portfolio serves as a comprehensive showcase of my journey through cloud engineering, highlighting hands-on experience with AWS, data pipelines, DevOps practices, and full-stack development. Designed with recruiters and hiring managers in mind, particularly those in fintech and banking sectors.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Clean UI/UX**: Professional layout with intuitive navigation
- **Project Showcase**: Detailed documentation of bootcamp projects with screenshots and links
- **Skills Matrix**: Comprehensive display of technical competencies across cloud, DevOps, data, and development
- **Contact Integration**: Direct links to GitHub, LinkedIn, and professional email
- **Fast Loading**: Optimized assets and Vercel CDN for quick page loads
- **Automated Deployment**: Continuous deployment from GitHub to Vercel

## Technologies Used

### Frontend

- HTML5 (semantic markup)
- CSS3 (Flexbox, Grid, responsive design)
- JavaScript (vanilla JS for interactivity)

### Deployment & DevOps

- Git & GitHub (version control)
- Vercel (hosting)

### Design Principles

- Mobile-first responsive design
- Accessibility best practices

## Local Development

### Prerequisites

- A modern web browser
- Git installed on your machine
- A code editor (VS Code recommended)

### Installation

1. Clone the repository

```bash
git clone https://github.com/Khaya14/Professional-Personal-Portfolio.git
```

2. Navigate to project directory

```bash
cd professional-personal-portfolio
```

3. Open in browser

```bash
# Simply open index.html in your browser
# Or use a local server (recommended)
```

### Using Live Server (VS Code)

1. Install the Live Server extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

## Deployment

This project is deployed on Vercel with automatic deployments from the main branch.
